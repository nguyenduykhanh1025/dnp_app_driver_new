Column offset
  • Options1
  • Options2

Use the .col-md-offset-* class to offset the column to the right. These classes actually use the * selector to add the left margin to the current element. For example, the .col-md-offset-4 class shifts the .col-md-4 element to the right by 4 columns (column) width.

.col-md-4
.col-md-4 .col-md-offset-4
.col-md-3 .col-md-offset-3
.col-md-3 .col-md-offset-3
.col-md-6 .col-md-offset-3
